# Mama Pizza Martigues - avis clients

Page statique en HTML, CSS et JavaScript vanilla pour recueillir les avis clients.

## Utilisation

Ouvrir directement `index.html` dans un navigateur, ou servir le dossier avec un serveur statique simple, par exemple :

```bash
python3 -m http.server 8000
```

Puis visiter `http://localhost:8000`.

Les notes de 4 ou 5 étoiles redirigent vers la page Google fournie. Les notes de 1 à 3 étoiles affichent un formulaire local sans envoi réel à un serveur.
